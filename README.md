# DexHTML Gym Landing

## AI-assisted landing page experiment

DexHTML Gym Landing — адаптивный неоновый лендинг для фитнес-тематики.

Проект создан как эксперимент с визуальным стилем, тёмной темой, неоновыми акцентами, анимациями и статической публикацией.

---

## Demo

[Посмотреть онлайн](https://dexhtml.github.io/gym-landing-dexhtml/)

---

## Что проверялось

- тёмная визуальная тема;
- неоновые акценты;
- адаптивность от десктопа до мобильных;
- плавные появления и hover-эффекты;
- SVG-иконки в едином стиле;
- плавная прокрутка по якорям;
- публикация через GitHub Pages.

## Статус

**Experimental / demo project**

Этот репозиторий не является основным направлением профиля. Он оставлен как часть архива экспериментов с лендингами, визуальной подачей, адаптивностью и AI-assisted workflow.

Основной фокус профиля сейчас — внутренние цифровые инструменты, сервисные системы, учёт техники, заявки, графики и AI-assisted проекты.
