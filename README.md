# DexHTML Gym Landing

## Responsive fitness landing page

DexHTML Gym Landing — адаптивный лендинг для фитнес-тематики с тёмной визуальной темой, неоновыми акцентами и анимациями.

Проект создан как демонстрация визуального frontend-дизайна, адаптивной вёрстки и статической публикации через GitHub Pages.

---

## Demo

[Посмотреть онлайн](https://dexhtml.github.io/gym-landing-dexhtml/)

---

## Что реализовано

- тёмная визуальная тема;
- неоновые акценты;
- адаптивность от десктопа до мобильных устройств;
- плавные появления элементов;
- hover-эффекты;
- SVG-иконки в едином стиле;
- плавная прокрутка по якорям;
- публикация через GitHub Pages.

---

## Что показывает проект

- работу с визуальной концепцией лендинга;
- адаптивную компоновку блоков;
- использование CSS-анимаций и интерактивных состояний;
- организацию статического frontend-проекта;
- подготовку сайта к публикации через GitHub Pages.
